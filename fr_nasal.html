鼻母音とリエゾン
フランス語らしい響きの要点 ・ Nasals & liaison
フランス語の発音でカギになるのが、鼻母音（鼻に抜ける母音）とリエゾン（単語がつながる連音）です。さらに語末の字を読まないことも多く、つづりと音のずれが大きい言語。要点を音（▶）で確かめましょう。
鼻母音（4つ）Nasal vowels
母音のうしろに n・m があり、そこで音節が終わると、その母音が鼻に抜けます（n・m は「ヌ・ム」とは読みません）。代表的な4つ：
つづり読みの目安例（意味）音
an / enアン（鼻）grand（大きい）
on / omオン（鼻）bon（よい）
in / ainアン（口を横に・鼻）pain（パン）
unアン（鼻）un（1つ）
例：enfant（アンファン＝子ども）は鼻母音が2つ。
鼻に抜く ⇄ 抜かないNasal vs oral
n・m のうしろに母音が続く（または n が重なる）と、鼻に抜けず、n・m をはっきり読みます。男性形・女性形のペアで違いがよく出ます。
bon / bonne
よい（男・鼻） / よい（女・nを読む）
un / une
1つ（男・鼻） / 1つ（女・nを読む）
plein / pleine
満ちた（男・鼻） / 満ちた（女・nを読む）
リエゾン（連音）Liaison
ふだんは読まない語末の子音が、次の語が母音で始まるとつながって発音されます。これがフランス語のなめらかな響きを作ります。
つづりつながる音意味音
les amisレザミ（s→z）友だち（複）
vous avezヴザヴェ（s→z）あなたは持つ
un petit ami…プティタミ（t）ボーイフレンド
nous avonsヌザヴォン（s→z）私たちは持つ
語末の字は読まないことが多いSilent endings
語末の e・s・t・d・x などはふつう読みません。だからつづりより音は短くなります。
つづり読み意味音
petitプティ（t読まない）小さい
grandグラン（d読まない）大きい
beaucoupボクー（p読まない）たくさん
vousヴ（s読まない）あなた（たち）
日本語話者へのヒントTips
・鼻母音は、日本語の「ん」を言い切らず、母音と同時に鼻へ抜くイメージ。n・m を「ヌ・ム」と読まないのがコツ。
・un / une のように、n のうしろに母音（や e）が来ると鼻に抜けず n を読みます。ペアで覚えましょう。
・リエゾンは「次が母音なら、隠れていた子音が顔を出す」と考えると分かりやすいです（s は z の音に）。
・つづりの語末の字は読まないことが多い。単語は「音」で覚えるのが近道です。
発音は端末の音声エンジン（フランス語）を使用。カナ表記は読みの目安です。

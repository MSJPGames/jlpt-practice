発音のポイント
英語の音・日本語話者の弱点 ・ English pronunciation
英語は音の数が日本語より多く、日本語にない音や、強弱リズムがあります。日本語話者がつまずきやすい所を、聞き比べ（▶）で確認しましょう。
L と RL / R
L＝舌先を上の歯ぐきにつけて離す。R＝舌を口の中でどこにもつけず、少し丸める。日本語のラ行は両者の中間なので、意識して区別を。
light / right
[laɪt] / [raɪt] ・ 光 / 右
rice / lice
[raɪs] / [laɪs] ・ 米 / しらみ
play / pray
[pleɪ] / [preɪ] ・ 遊ぶ / 祈る
th（θ / ð）th sounds
舌先を上下の歯の間に軽くはさんで息を出す音。無声＝θ（think）、有声＝ð（this）。日本語の「サ・ザ」や「ス」で代用しがちなので注意。
think / sink
[θɪŋk] / [sɪŋk] ・ 考える / 沈む
this / that
[ðɪs] / [ðæt] ・ これ / あれ
母音の区別Vowels
日本語の母音は5つ。英語はもっと多く、長さだけでなく口の開きもちがいます。似た母音のペアを聞き分けましょう。
sit / seat
[sɪt] / [siːt] ・ 座る / 席
cat / cut
[kæt] / [kʌt] ・ 猫 / 切る
full / fool
[fʊl] / [fuːl] ・ 満ちた / ばか
強勢（アクセント）とリズムStress & rhythm
英語は強く長く読む音節と弱く短い音節のリズムで話します（日本語は高低・等間隔）。強勢の位置で品詞や意味が変わることも。弱い母音はあいまい母音 ə（schwa）になります。
語強い所意味音
PREsent（名）前贈り物
preSENT（動）後ろ提示する
baNAna真ん中バナナ（ba-na-na）
about後ろa＝ə（弱い）
日本語話者へのヒントTips
・カタカナ発音（母音を足す）に注意。strike は「ストライク」ではなく1音節[straɪk]。子音が続く所に母音を入れない。
・語尾の子音（-t, -k, -d）をはっきり。日本語のように母音で終わらせない。
・文は強い語（内容語）を強く、弱い語（the, to, of）を弱く。このリズムが「英語らしさ」になります。
発音は端末の音声エンジン（英語）を使用。

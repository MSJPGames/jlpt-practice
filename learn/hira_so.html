so
[so]
1画
成り立ち（なりたち）
楷書曽
行書筆で
草書くずす
ひらがなそ
漢字「曽」をくずして生まれた字です
発音
「so」と発音します。
この字を使う言葉
そら（空）sky
そふ（祖父）grandfather
書き順
上のアニメーションが筆順です（1画）。「もう一度」で何度でも再生できます。基本は上から下・左から右。
なぞって練習
さいしょの丸から、指でなぞってね

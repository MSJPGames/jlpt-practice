🗣️ 会話練習（授業用）

授業用ページです。合言葉を入れて使います。場面や教科書の課を選んで練習できます。

このページは授業用です。合言葉を知っている人だけが使えます。
キーはこの端末の中だけに保存され、外部には送られません（Google以外）。 取り方: aistudio.google.com/apikey でログイン →「Create API key」→ コピーして貼り付け（無料）。
『みんなの日本語 初級Ⅰ・Ⅱ』全50課の練習C（やりとり）から選べます。相手役が、その練習の目標に合わせて会話をリードします。
オフのときは、相手は必ず日本語だけで話します。オンにすると、英語まじりの相手への対応を練習できます（接客などの場面向け）。
教科書の進度に合わせて、相手役が使う言葉をやさしくします。いちばん正確なのは「この課までの範囲」に実際の語彙・文法リストを貼る方法です。
声の設定（より自然な声にする）
「Gemini 自然音声」は、今のキーのまま、より人間らしい声で読み上げます（試験的機能。使えないときは自動でブラウザの声に切り替わります）。1回の返事で消費が少し増えます。
端末によって声の種類が違います。「Google 日本語」「Microsoft ... Natural」「Kyoko」などがあれば、それを選ぶと自然になります。
VOICEVOX はより自然で表情のある声が出せますが、多人数で使うにはエンジンをネット上に用意（CORS許可）する必要があります。 利用時は規約にしたがい話者のクレジット表記が必要です。うまく鳴らないときは自動でブラウザの声に切り替わります。
ヒント: 間違えても大丈夫。まずは最後まで会話を続けてみましょう。

※ 音声認識は Chrome（パソコン・Android）で最もよく動きます。iPhone など声がうまく使えない場合は、下の入力欄に文字で送信すれば同じように練習できます。
